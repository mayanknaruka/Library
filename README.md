Library
live demo : 
A simple library application built with HTML, CSS, and JavaScript as part of The Odin Project.

Features
Add new books to the library
Display books as cards
Remove books from the library
Toggle a book's read status
Store all book objects in an array
Each book has a unique ID generated using crypto.randomUUID()
Built With
HTML
CSS
JavaScript
How to Use
Click the New Book button.
Enter the book details.
Submit the form to add the book.
Use Toggle Read to change the read status.
Use Remove to delete a book from the library.
